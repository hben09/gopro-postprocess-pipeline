## GoPro Labs Settings

| Command | Value | Description              |
|---------|-------|--------------------------|
| BITR    | 180   | Bitrate set to 180 Mbps                          |
| DAMP    | 3     | Auto-exposure damping (3× slower than default)    |
| EXPX    | 160   | Max exposure time (slowest shutter: 1/160)        |
| NR01    | 1     | Noise reduction at 1% (essentially minimal)       |
| GCMP    | 1     | Disable gyro-compensated exposure                 |
| HIST    | 5     | Display histogram overlay (contrast level 5)      |
| HSTP    | 1,90,42 | Histogram position (x:1, y:90) and size (42)    |
| EXPS    | 1     | Display current ISO and shutter speed on screen   |
